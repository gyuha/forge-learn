# 2026-07-01 — 내 목소리 클론으로 사용법 영상 재더빙 + 재사용 파이프라인

## 계획 vs 실제
- 계획대로 된 것: 32슬라이드 클론 합성(19분 분량, RTF ~0.37 — M5 실용적), 재타이밍·재합성·SRT 생성, 원본 미변경, 재사용 파이프라인(`tts/run.sh`) 완성. ADR-0001(로컬 전용) 준수.
- 계획과 달랐던 것:
  1. **참조 길이 강제 제약**: 계획은 "≤10초, 필요 시 정규화"였으나 GPT-SoVITS(`TTS.py:817`)가 3~10초를 코드로 강제 → 정규화가 선택이 아닌 **필수**. 19.9초 참조를 무음 경계(8.96초)로 트림.
  2. whisper 한국어 전사 오인식 → 절단 위치 산출에만 활용, prompt_text는 원본 전사 수기 교정.
  3. 음량 낮음(mean -30dB) → UAT 후 -16 LUFS 부스트 재믹스.

## 학습
- 다음에 다르게 할 것:
  1. **GPT-SoVITS 참조 클립은 처음부터 3~10초로 준비** — 코드 강제 제약이라 정규화 여부와 무관하게 입력 사양에 명시. "≤10초"가 아니라 "3~10초"가 정확한 범위.
  2. **최종 믹스 음량 체크를 파이프라인에 포함** — amix 기본 정규화로 전체가 낮아지므로, 렌더 후 loudnorm 통과 확인 또는 기본으로 -16 LUFS 정규화 단계 추가.
