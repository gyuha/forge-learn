# 음성 클론 백엔드는 로컬 GPT-SoVITS — 목소리 외부 전송·자동 클라우드 폴백 금지

forge 교육 영상의 내레이션을 사용자 본인 목소리로 재더빙하기 위한 음성 클론 백엔드로, 상용 API(Typecast·ElevenLabs)가 한국어 품질·편의·환경 회피 면에서 더 안전하다는 그릴링 권고에도 불구하고 **로컬 오픈소스 GPT-SoVITS**(MIT, 한국어 공식 지원, 5초 zero-shot)를 선택했다. 이유는 **프라이버시(목소리가 기계 밖으로 나가지 않음)와 비용 0원**이다. 그 대가로 Apple M5에서의 GPU(MPS) 가속 추론 속도가 미검증이며(CPU 폴백 시 32개 합성이 매우 느릴 수 있음), 이 저장소가 과거 두 번 외부 설치 게이트에 막힌 전력이 있어 torch·멀티 GB 모델 설치 실패 위험을 안는다.

**핵심 제약(코드에 안 보이는 데이터 경계):** 로컬이 설치 게이트에 막히거나 M5에서 쓸 수 없을 만큼 느릴 경우, **로컬 완화책(CPU 모드·소형 모델·스텝 감축)을 소진한 뒤 중단·보고하며, 유료 클라우드로 자동 전환하지 않는다.** 자동 전환은 로컬을 택한 이유(프라이버시·비용)를 무효화하기 때문이다. 클라우드 전환은 사람이 명시적으로 결정할 때만 한다.

## Considered Options
- **Typecast** (한국 토종, 한국어 1순위, 진짜 클로닝 API + Python SDK, ₩39,000/월) — 재사용 파이프라인·한국어엔 최적이나 목소리가 클라우드로 전송되고 학습 사용 약관 미확인.
- **ElevenLabs** (가장 빠름, 1~2분 참조) — 비-Enterprise 등급에서 목소리가 **default-on으로 모델 학습에 사용**되고 opt-out이 소급 안 됨. 프라이버시 비용 큼.
- **GPT-SoVITS 로컬** (선택) — 무료·완전 로컬·MIT·한국어 공식. M5 속도 미검증·설치 게이트 리스크 감수.

## Consequences
- 음성 백엔드는 `synthesize(text) -> audio` 어댑터 한 겹으로 격리해 향후 교체 비용을 낮춘다(현재는 단일 백엔드, 과설계 금지).
- 입문 영상 재더빙·fine-tune 승급 등 후속 작업도 이 제약(로컬 전용·자동 클라우드 금지)을 물려받는다.
